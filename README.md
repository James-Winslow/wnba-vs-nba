# WNBA vs. NBA Shooting Accuracy Analysis
<img src="images/header.png" alt="WNBA vs NBA" width="600"/>

## Project Overview
This project provides an in-depth comparison of shooting accuracy between WNBA and NBA players, focusing on three key metrics: Field Goal Percentage (FG%), Free Throw Percentage (FT%), and Three-Point Percentage (3P%). The analysis involves scraping, cleaning, and statistically analyzing the data to uncover meaningful insights. This project demonstrates a wide range of skills that are crucial for data-driven decision-making in sports analytics and beyond.

## Skills Demonstrated
- **Data Scraping**: Automated data collection using `requests` and `BeautifulSoup` to scrape player statistics directly from online sources, including robust error handling and edge case management.
- **Data Cleaning**: String manipulation to clean and split concatenated strings, convert string percentages to numeric formats, and handle missing data.
- **Statistical Analysis**:
  - Normality Testing: Used the Shapiro-Wilk test to assess the normality of shooting percentages, highlighting significant deviations.
  - Variance Homogeneity Testing: Conducted Levene’s test to evaluate the equality of variances between the leagues.
  - T-tests and Non-parametric Tests: Performed independent T-tests and Mann-Whitney U tests, supplemented by effect size calculations (Cohen’s d).
- **Data Visualization**:
  - Descriptive Plots: Created diagnostic plots to visualize the distribution of shooting percentages.
  - Comparative Box Plots: Developed box plots to visually compare shooting accuracies between the leagues, with annotated statistical test results.
  - Aesthetic Design: Applied Seaborn’s design elements for clear and engaging visualizations.
- **Python Proficiency**: Extensive use of Pandas for data manipulation, and SciPy for statistical analysis, showcasing advanced data handling and analysis skills.
- **Version Control & Collaboration**:
  - Incremental Development with Git: Maintained a clean and organized Git commit history with clear documentation.
  - Collaboration Readiness: The project is structured and documented to facilitate easy collaboration.

## Current Analysis Results
- **Normality Testing**:
  - WNBA FG%: Non-normal distribution (Shapiro-Wilk p-value: 1.25e-08)
  - NBA FG%: Non-normal distribution (Shapiro-Wilk p-value: 2.83e-14)
  - WNBA FT%: Non-normal distribution (Shapiro-Wilk p-value: 1.07e-12)
  - NBA FT%: Non-normal distribution (Shapiro-Wilk p-value: 1.01e-17)
  - WNBA 3P%: Non-normal distribution (Shapiro-Wilk p-value: 1.09e-11)
  - NBA 3P%: Non-normal distribution (Shapiro-Wilk p-value: 6.93e-25)
- **Variance Homogeneity**:
  - Significant differences in variances were found for all shooting percentages between WNBA and NBA players, as indicated by the Levene's test results.
- **T-tests and Effect Sizes**:
  - Field Goal %: T-test p-value: 8.26e-37, Cohen's d: 2.63 (Large effect)
  - Free Throw %: T-test p-value: 4.20e-11, Cohen's d: 1.12 (Large effect)
  - Three Point %: T-test p-value: 3.51e-21, Cohen's d: 1.74 (Large effect)
- **Visualization Insights**:
  - The NBA data shows significantly more condensed distributions across all shooting percentages compared to the WNBA. This finding suggests potentially different factors influencing shooting accuracy in the two leagues, warranting further investigation.
